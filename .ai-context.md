This is a VS Code Extension built using:
- TypeScript
- esbuild bundler
- npm

Extension Name: flutter-fahhh-architect

Goal:
Generate Clean Architecture Flutter apps with a humorous Fahhh twist.

Features required:
1. Command to create Flutter project
2. Copy architecture template into lib/
3. Support command for Buy Me a Coffee
4. Proper activation events
5. Clean error handling
6. Console humor logs
7. Production-ready structure

We are using esbuild.
Do NOT break bundler configuration.
Use fs-extra for file operations.
