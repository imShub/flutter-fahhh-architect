# Flutter Fahhh Architect — MVVM (Provider)

This is a **lib-only** template to be copied into a freshly-created Flutter app (after `flutter create <appName>`).

Contains only:
- `lib/`
- `pubspec.yaml` (merge snippet)
- `README.md`

## What you get

- MVVM structure (Model / View / ViewModel)
- Provider state management
- Dio API abstraction
- audioplayers sound service
- BottomNavigationBar with 4 screens
- Theme toggle
- Named routes
- Responsive helper
- Fahhh counter demo (sarcasm + meme SnackBars + random sound)
- Buy Me a Coffee support link in README + Settings

## How to use

1. Create an app:

```bash
flutter create my_app
```

2. Replace only the generated `lib/` with this template’s `lib/`.
3. Merge dependencies/assets from this template’s `pubspec.yaml` into your project `pubspec.yaml`.
4. Run:

```bash
flutter pub get
```

## Assets you must add

Add these files in your Flutter app:

- `assets/sounds/fahhh.mp3`
- `assets/sounds/bruh.mp3`
